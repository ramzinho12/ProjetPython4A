# Puissance 4 en Python : Un Guide Complet

Ce guide détaillé vous présente la réalisation d'un jeu Puissance 4 en Python, en s'appuyant sur le langage Python et la bibliothèque Pygame pour une expérience utilisateur graphique enrichie.

## Prérequis

Avant de plonger dans l'aventure, veillez à :

1. Avoir Python installé sur votre machine.
2. **Configuration des ressources graphiques :** Le projet contient des images essentielles (image3, image4, image5) dans son répertoire. Ajustez les chemins d'accès dans le script `PROJET PYTHON PUISSANCE 4.py` comme suit :

    ```python
    background_image = pygame.image.load("votre/chemin/vers/image3")
    ```

    Remplacez "votre/chemin/vers/image3" par le chemin précis de l'image sur votre système.

3. **Lancement de l'Application :** Ouvrez `code_serveur.py` et `code_client.py`. Démarrez le serveur avant de lancer le client pour assurer une communication fluide entre les deux.

## Fonctionnalités Clés

- **Mode Joueur vs IA :** Affrontez une intelligence artificielle sophistiquée, capable de stratégies de jeu avancées.
- **Mode Joueur vs Joueur :** Partagez le clavier pour une partie entre amis dans ce mode compétitif.
- **Interface Utilisateur Graphique :** Profitez d'une interface agréable et intuitive conçue avec Pygame.
- **Tests Unitaires :** Le projet inclut des tests unitaires pour garantir la robustesse et la fiabilité des composants clés.

## Architecture du Projet

Le cœur du jeu est concentré dans le fichier **PROJET PYTHON PUISSANCE 4.py**, où toute la magie opère.

## Documentation et Support

Chaque fonction et méthode est soigneusement documentée dans le code grâce aux docstrings, facilitant ainsi la compréhension et l'extension du projet.
